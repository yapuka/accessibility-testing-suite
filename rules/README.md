# Règles d'accessibilité

Ce dossier rassemble les règles de référence par norme :

- WCAG 2.2
- RGAA
- EN 301 549
- European Accessibility Act

Chaque fichier doit contenir :

- une définition de la règle,
- des critères d'acceptation,
- un exemple de bonne implémentation,
- un exemple de mauvaise implémentation.
