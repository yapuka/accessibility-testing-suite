# WCAG 2.2 - Non-text Content

- Description : fournir une alternative textuelle pertinente pour les éléments non textuels.
- Acceptance criteria :
  - une image informative possède un attribut `alt` descriptif ;
  - une image décorative est masquée ou ignorée par l'assistance technique.
